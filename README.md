# Privy Snipe-IT Asset Management

Download Docker and set up your own .env file using the .env.example file

Copy the .env.example file and rename the copy to .env and follow the steps in the file to add the required information.

Start the Docker container, run through the setup on localhost:8000 and finall import data via the automation script.
Then, install Python 3.14 or above if you want to use the API (you must first generate yourself an API key in Snipe-IT and
add it to your .env)
